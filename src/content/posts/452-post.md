<p>고기 굽기 할 때 혹은 레스토랑에서 주문 할 때 고기를 어느정도 익힐지 물어보는 경우가 있습니다.</p>
<p>어릴 때 20살 초반에는 잘 몰르지만 모른다고 하면 창피한것 같은 생각에 아는 단어를 빨리 말해 버렸는데</p>
<p>그게 '레어' 였어요.</p>
<p>지금이라면 맛있게 잘 먹겠지만 그때는 피가 뚝뚝 떨어져서 '아 망했다'라고 생각했던 기억이 있어요.</p>

<p>저처럼 아직 고기 굽기 정도에 대한 용어를 잘 모르시는 분들을 위해서 정리를 해 보았습니다. </p>

<h2><b>고기 익힘 정도 </b></h2>

<h2><b>블루레어 고기 상태 설명</b></h2>

<ul style="list-style-type: disc;">
<li>겉에만 살짝 지지고 옆쪽과 안쪽은 완전히 붉은 상태</li>
<li>붉은색은 익지 않은 생고기 상태라고 볼수 있다.</li>
<li>스테이크를 먹으러가는 레스토랑에서는 보통 잘 안먹는다. </li>
</ul>


<h2><b>레어<b> 고기 상태 설명</b></b></h2>

<ul style="list-style-type: disc;">
<li>안쪽에 75% 붉은 붉은 빛이 돈다.</li>
<li>온도가 낮게 조리가 된거지 레어가 생고기도 아니고 육즙이 핏물도 아니라고 한다.</li>
<li>고기 중안 부분 온도 - 49~53도 </li>
<li>* 레어를 처음 먹어보는 사람은 미디엄이나 미디엄레어를 먼저 먹어보고 차차 내려가 보도록 하자</li>
</ul>


<h2><b>미디엄 레어<b> 고기 상태 설명</b></b></h2>

<ul style="list-style-type: disc;">
<li>안에 50% 붉은 혹은 핑크 빛이 돈다. </li>
<li>육즙이 많습니다.</li>
<li>고기 중안 부분 온도 - 54~58도 </li>
</ul>

<h2><b>미디엄<b> 고기 상태 설명</b></b></h2>

<ul style="list-style-type: disc;">
<li>안쪽에 25%정도 핑크 빛이 돈다.</li>
<li>고기 중안 부분 온도 - 59~62도</li>
<li>최근 한국인이 선호하는 상태라고 한다.<br/><br/></li>
</ul>
<h2><b>미디엄 웰던 고기 상태 설명</b></h2>

<ul style="list-style-type: disc;">
<li>아주 얇게 핑크 빛이 돈다</li>
<li>고기 중안 부분 온도 - 63~66도</li>
<li>전통적으로 한국인이 선호하는 상태라고 한다.</li>
</ul>

<h2><b>웰던 고기 상태 설명</b></h2>

<ul style="list-style-type: disc;">
<li>핑크 빛이 없게 완전히 익힌 상태</li>
<li>고기 부위나 고기 등급에 따라 너무 익히면 고기가 딱딱해진다.</li>
<li>고기 중안 부분 온도 - 67~68도</li>
</ul>
