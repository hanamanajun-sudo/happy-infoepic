<hr contenteditable="false">
<h2 style="text-align: center;"><b><span style="color: #333333;">근로장려금 반기  <span style="background-color: #9feec3;"><u>신청자격 조회방법</u></span></span></b></h2>
<hr contenteditable="false"/>
<h3><span style="background-color: #f6e199;"><b><span style="color: #333333;"><span style="color: #333333;">ARS </span>신청자격 조회방법</span></b></span></h3>

<p>1. 1544-9944 국세청 ARS 전화를 건다.</p>

<p>2. 2번 다이얼을 눌러 '안내대상자 확인'를 선택한다.</p>

<p>3. 본인의 주민등록번호 + # 버튼</p>

<p>4. 본인의 휴대폰 번호 + # 버튼</p>

<p>5. 이후 자격 조건 안내 문자 발송된 것을 확인한다.</p>


<h3><span style="background-color: #f6e199;"><b><span style="color: #333333;"><span style="color: #333333;">홈택스 앱 </span>신청자격 조회방법</span></b></span></h3>

<p>1. 로그인</p>

<p>2. 상단 메뉴 중 '신청/제출' 선택</p>

<p>3. 근로장려금(반기) 선택</p>

<p>4. 안내대상자 여부 조회 클릭</p>

<h2 style="text-align: center;"><b><span style="color: #333333;"><b><span style="color: #333333;">근로장려금 반기  </span></b><u><span style="background-color: #9feec3;">신청방법</span></u></span></b></h2>
<hr contenteditable="false"/>
<h3><span style="background-color: #99cefa;"><b><span style="color: #333333;">신청안내문(개별인증번호)을 받은 경우</span></b></span></h3>

<h4><span style="color: #ef5369;">1.<b>ARS 전화(보이는ㆍ음성)</b></span></h4>

<p>신청안내 대상자가 안내문 등에서 확인한 개별인증번호를 활용하여</p>
<p>간편하게 신청할 수 있습니다</p>

<p id="">* <b>1544-9944</b> → 장려금 1번 → 주민등록번호 13자리 → </p>
<p>발신번호 미일치시(개별인증번호 8자리 입력) → </p>
<p>동의하고 신청 1번 → 연락처 및 계좌번호 등록 확인 → 신청완료</p>

<h4><span style="color: #ef5369;">2.<b>손택스(모바일 앱)</b></span></h4>
<p>신청안내대상자가 안내문 등에서 확인한 개별인증번호를 활용하여</p>
<p>간편하게 신청할 수 있으며, 먼저 「국세청 홈택스(손택스) 앱」을 다운받아</p>
<p>설치하여야 합니다.</p>

<p id="">* 손택스 앱 실행 → 근로장려금 반기신청 → </p>
<p>주민등록번호 뒤 7자리와 개별인증번호 입력 → </p>
<p>계좌번호·연락처 입력 → 신청하기</p>

<h4><span style="color: #ef5369;"><b>3.홈택스</b></span></h4>
<p>신청안내 대상자가 국세청 홈택스(www.hometax.go.kr)에 로그인하여</p>
<p>간편하게 신청가능합니다.</p>

<p id="">* 국세청홈택스 로그인 → 신청·제출 → </p>
<p>근로장려금 반기신청 → 간편신청하기 → </p>
<p>신청요건 확인 → 계좌번호·연락처 입력 → 신청하기</p>

<h4><b><span style="color: #ef5369;">4.「장려금 전용 전화상담센터」에 전화하여 대리신청을 요청</span></b><br/>→ 코로나19 감염 예방을 위해 2020년 9.1.~9.15.동안 한시적 운영</h4>

<p><b>지방청별 장려금전용 전화상담센터 (서울청 02-2114-2199, 중부청 031-888-4199, 부산청 051-750-7199, 인천청 032-718-6199, 대전청 042-615-2199, 광주청 062-236-7199, 대구청 053-661-7199)</b></p>

<hr contenteditable="false"/>
<h3><span style="background-color: #99cefa;"><b>신청안내문을 받지 못한 경우</b></span></h3>

<p>소득, 재산 요건 등 신청요건 확인 후 인터넷 또는 세무서 방문하여 서면으로 신청</p>

<p id="">* 국세청홈택스 로그인 → 신청·제출 → </p>
<p>근로장려금 반기신청 → 일반신청하기 → </p>
<p>신청요건 확인 → 인적사항 작성 → 소득명세 작성 → </p>
<p>신청자격 확인 → 계좌번호·연락처 입력 → </p>
<p>신청확인 및 전송 → 접수증 출력</p>
